## рҹӨ– Agent System

Each agent is a **robot character** with its own color identity:

| Agent Role | Color | Model | Purpose |
|:---:|:---:|:---:|:---|
| рҹ‘‘ **Lead** | рҹҹЎ Gold | `claude-3-5-sonnet` | Coordinates and reviews work |
| рҹ”Қ **Researcher** | рҹ”ө Blue | `claude-3-5-sonnet` | Analyzes data and provides insights |
| вҡҷпёҸ **Worker** | рҹҹў Green | `gpt-4o` | Executes tasks efficiently |
| рҹҺҜ **Critic** | рҹ©· Pink | `claude-3-5-sonnet` | Reviews and finds issues |
| вңҚпёҸ **Writer** | рҹҹЈ Purple | `gpt-4o` | Creates documentation |

Agents sit at **desks with monitors, keyboards, coffee mugs, and pencils**. They **bob and swing their arms** when thinking, and a **status orb** floats above each one:

- вҡӘ **Grey** = Idle
- рҹ”ө **Blue** = Thinking
- рҹҹў **Green** = Done
- рҹ”ҙ **Red** = Error

## рҹҺӯ Mock Pipeline Demo

The pipeline runner simulates agent activity with **fake token streaming**:

```
в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ    вҡЎ wire    в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ
в”Ӯ рҹ‘‘ Lead    в”Ӯ в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв–ё в”Ӯ вҡҷпёҸ Worker  в”Ӯ
в”Ӯ thinking.. в”Ӯ   dot moves  в”Ӯ idle       в”Ӯ
в”Ӯ в–Ң         в”Ӯ              в”Ӯ            в”Ӯ
в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ              в””в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ
```

1. Click `в–¶ RUN ALL` to start the mock pipeline
2. Agents activate **sequentially** вҖ” Lead thinks first, then Worker
3. Each agent streams **character-by-character** output to the chat panel
4. Wires show **animated dots** traveling between connected agents
5. Status orbs change: вҡӘ вҶ’ рҹ”ө вҶ’ рҹҹў
